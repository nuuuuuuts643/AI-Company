## ハッシュタグ戦略

### 全ジャンル共通タグ（毎回使用）
- `#flotopic`
- `#AIニュース`
- `#ニュースまとめ`

### ジャンル別タグ

| ジャンル | メインタグ | サブタグ |
|---|---|---|
| 政治・国内 | `#政治` `#国内ニュース` | `#日本政治` `#国会` |
| 経済・ビジネス | `#経済` `#ビジネス` | `#株式市場` `#マーケット` |
| テクノロジー | `#テクノロジー` `#AI` | `#スタートアップ` `#DX` |
| 国際・海外 | `#国際ニュース` `#海外` | `#世界情勢` `#外交` |
| スポーツ | `#スポーツ` | `#サッカー` `#野球`（話題に応じて） |
| エンタメ | `#エンタメ` `#芸能` | `#映画` `#音楽` |
| 科学・健康 | `#科学` `#健康` | `#医療` `#環境` |

### 投稿頻度の目安
- 自動ツイート（日次ハイライト）: 毎日1回
- 手動ツイート（話題の急騰時）: 週2〜3回
- リプライ・エンゲージメント: 随時

- `@nhk_news`
