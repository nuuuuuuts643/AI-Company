# セッションプロンプト: T2026-0501-K keyPoint 品質改善

> 起動トリガー: 「K起動」

## 品質の前提認識（重要）

**100字 = 最低条件。品質ではない。**

keyPoint の本来の目的は「読者がそのトピックの続きを読みたくなること」。
ジャンル別に読者とその期待が違う:

- **エンタメ**: 芸能人/有名人絡み → 感情的引っかかり（驚き・ドラマ・意外性）
- **テクノロジー**: 企業/技術発表 → 自分への影響・業界の変化の意味
- **政治/国際**: → 自分の生活・社会への波及
- **経済/ビジネス**: → お金・市場・企業動向の実質的意味

「当たり障りのない要約 100字以上」は失敗。100字あっても読まれない。
実機（flotopic.com）で見て「続きを読みたくなるか」を人間が評価するのが唯一の正解判断。
